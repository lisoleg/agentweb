## 平台简介

AgentWeb Sigma Cloud 是一个基于 **信息几何与意识场统一理论** 的新一代去中心化社交网络平台。

### 核心理念

平台将用户在信息空间中的行为量化为 **Φ 场（整合信息量）**，通过四令牌系统激励高质量内容创作与计算资源共享。

### 主要特性一览

| 特性 | 说明 |
|------|------|
| 🔐 无缝 DID | 设备指纹 + 生物识别，无感登录 |
| 🌐 联邦社交 | 接入 Fediverse，与 Mastodon/Pixelfed 等互通 |
| 🧠 φ 引擎推荐 | 基于 Φ 场强度的智能内容推荐 |
| 🪙 四令牌激励 | Calc/Wit/Word/Pass 多维度奖励体系 |
| ⚡ FPGA 加速 | 硬件级 Φ 场计算加速 |
| 🔗 Web3 集成 | 连接加密钱包，链上令牌管理 |

## 快速上手

### 第一步：访问平台

在浏览器中打开平台地址（由管理员提供，默认本地部署为 `http://localhost:3000`）。

### 第二步：注册账号

1. 点击首页右上角 **「注册」** 按钮
2. 输入用户名、邮箱、密码
3. 可选：启用生物识别（指纹/面容）实现无缝登录
4. 点击 **「创建账号」**
5. 检查邮箱并完成验证（如已配置邮件服务）

### 第三步：完善个人资料

1. 点击右上角头像 → **「个人设置」**
2. 上传头像与封面图
3. 填写个人简介
4. 设置个性化域名（用于联邦身份，如 `@yourname@agentweb.social`）

### 第四步：关注推荐账号

首次登录后，φ 引擎会根据你的兴趣推荐首批关注账号。可以直接关注，也可以跳过。

## 注册与登录

### 注册新账号

访问注册页面 `http://localhost:3000/register`（或点击首页「注册」按钮）。

填写以下信息：

| 字段 | 必填 | 说明 |
|------|------|------|
| 用户名 | ✅ | 3-20 字符，仅支持字母数字下划线 |
| 邮箱 | ✅ | 用于验证与通知 |
| 密码 | ✅ | 至少 8 位，建议包含大小写+数字 |
| 邀请码 | ⬜ | 若平台开启邀请制则必填 |

### 无缝 DID 登录

注册时勾选 **「启用无缝登录」**，系统将：

1. 采集设备指纹（硬件特征，不可逆哈希）
2. 可选采集生物识别信息（指纹/Windows Hello/面容 ID）
3. 自动生成 DID 密钥对（保存在本地安全存储）
4. 下次访问时自动登录，无需输入密码

> 🔒 你的私钥仅保存在本地设备中，平台无法访问。

### 登录方式

| 方式 | 说明 |
|------|------|
| 账号密码 | 传统用户名+密码登录 |
| DID 无缝 | 设备识别自动登录（需之前启用） |
| Web3 钱包 | 连接 MetaMask 等钱包，用钱包地址登录 |
| 联邦账号 | 使用其他 ActivityPub 平台账号登录（跨平台 SSO） |

### 忘记密码

1. 在登录页点击 **「忘记密码？」**
2. 输入注册邮箱
3. 查收重置密码邮件
4. 点击邮件链接，设置新密码

## 首页时间线

登录后默认看到 **首页时间线**，展示你关注的人发布的内容。

### 时间线类型

| 类型 | 说明 | 切换方式 |
|------|------|----------|

| 联邦 | 来自其他联邦平台（如 Mastodon）的内容 | 点击「联邦」标签 |

### 时间线操作

- **过滤**：按内容类型过滤（纯文字/图片/视频/链接）
- **排序**：切换「推荐排序」或「时间排序」

## 发帖与内容创建

### 发布新帖

1. 点击首页顶部 **「有什么新鲜事？」** 输入框
2. 输入文字内容（最多 5000 字符）
3. 可选添加：
   - 📷 图片（最多 4 张，支持 JPG/PNG/GIF/WebP）
   - 🎬 视频（最多 1 个，最大 100MB）
   - 🔗 链接（自动抓取预览卡片）
   - 📍 位置（可选共享地理位置）
   - 🏷️ 话题标签（以 `#` 开头，如 `#AgentWeb`）
4. 设置可见范围：
   - **公开**：所有人可见，包括联邦平台
   - **仅关注者**：仅粉丝可见
   - **仅自己**：草稿/私密笔记
   - **自定义**：指定特定用户可见
5. 点击 **「发布」**

### 内容编辑技巧

| 功能 | 语法 | 说明 |
|------|------|------|
| 粗体 | `**文字**` | 加粗显示 |
| 斜体 | `*文字*` | 斜体显示 |
| 代码 | `` `代码` `` | 行内代码 |
| 代码块 | ` ```语言\n代码\n``` ` | 多行代码高亮 |
| 引用 | `> 引用内容` | 引用块 |
| 链接 | `[文字](URL)` | 超链接 |
| 提及 | `@用户名` | 提及用户（自动补全） |

### 回复与引用

- **回复**：点击帖子下方的「回复」按钮，输入内容后发布
- **引用转发**：点击「转发」→「引用转发」，可附加评论后转发
- **纯转发**：点击「转发」→「直接转发」，无评论转发

### 编辑与删除

- 帖子发布后 **5 分钟内** 可编辑（点击帖子右上角 `⋯` →「编辑」）
- 任何时候均可删除（点击 `⋯` →「删除」）
- 删除后内容将从联邦平台同步删除（ActivityPub Delete 协议）

## 四令牌系统

AgentWeb 使用 **四令牌系统** 量化用户在平台中的贡献与价值。

### 令牌类型

```
┌─────────────────────────────────────────────────┐
│                四令牌体系                        │
├──────────┬──────────┬──────────┬───────────────┤
│   Calc   │   Wit    │  Word    │    Pass       │
│  计算令牌 │  智慧令牌 │ 语言令牌 │   通行证      │
│  🔮      │  💡      │  💬      │    🔑        │
└──────────┴──────────┴──────────┴───────────────┘
```

#### 1. Calc（计算令牌）🔮

- **含义**：奖励计算资源贡献
- **获取方式**：
  - 运行 FPGA 节点（硬件加速 Φ 场计算）
  - 运行全节点（维护网络）
  - 提供存储资源
- **消耗**：发布计算密集型内容（如 AI 生成内容）时消耗

#### 2. Wit（智慧令牌）💡

- **含义**：奖励高质量内容创作，权重由 Φ 场强度决定
- **获取方式**：
  - 发布高 Φ 值内容（被大量高质量互动）
  - 内容被管理员加精
  - 被 φ 引擎判定为「智慧内容」
- **消耗**：提升内容曝光权重（推广自己的内容）

#### 3. Word（语言令牌）💬

- **含义**：奖励社交互动贡献
- **获取方式**：
  - 发布帖子
  - 评论他人帖子
  - 转发优质内容
- **消耗**：某些高级功能（如置顶帖子、创建投票）

#### 4. Pass（通行证）🔑

- **含义**：身份与访问权限证明
- **获取方式**：
  - 完成 DID 注册
  - 完成 KYC 验证（如开启）
  - 长期活跃账号自动获得
- **消耗**：访问付费功能、创建联邦节点

### 查看令牌余额

1. 点击右上角头像 → **「我的钱包」**
2. 或访问 `http://localhost:3000/wallet`
3. 可查看四种令牌的当前余额与历史记录

### 令牌转账

1. 进入「我的钱包」→ 点击 **「转账」**
2. 输入对方的用户名或钱包地址
3. 选择令牌类型与数量
4. 确认交易（需连接 Web3 钱包签名）

> ⚠️ 令牌转账不可撤销，请仔细核对收款方。

## φ 引擎与内容推荐

### φ 引擎是什么？

φ 引擎是 AgentWeb 的 **智能推荐核心**，基于 Φ 场（整合信息量）理论，对内容与用户进行量化评估，实现：

- 个性化内容推荐
- 高质量内容识别
- 用户意识水平度量
- 联邦内容智能过滤

### Φ 值说明

每条内容都有一个 **Φ 值**（0~100+），代表其「整合信息量」：

| Φ 值范围 | 内容特征 | 推荐权重 |
|----------|----------|----------|
| 0~10 | 日常碎碎念 | 低 |
| 10~30 | 普通内容 | 正常 |
| 30~60 | 优质内容 | 高 |
| 60+ | 精品内容（加精） | 极高 |

### 提升个人 Φ 值

1. **发布深度内容**：长文、技术分析、原创研究
2. **获得高质量互动**：被高 Φ 值用户点赞/评论
3. **持续活跃**：规律发布，建立受众
4. **跨平台联邦**：与联邦平台用户互动，扩大影响范围

### 推荐算法透明度

φ 引擎的所有推荐逻辑均开源，可在 `phi-engine/` 目录查看源代码。

你可以在「设置」→「推荐偏好」中调整：
- 推荐内容类型权重（技术/艺术/社交/新闻）
- 是否过滤低 Φ 值内容
- 联邦内容推荐比例

## Web3 与区块链功能

### 连接 Web3 钱包

1. 点击右上角头像 → **「连接钱包」**
2. 选择钱包类型：
   - MetaMask（推荐）
   - WalletConnect（支持多种钱包）
   - Coinbase Wallet
3. 在钱包中确认连接
4. 连接成功后，头像旁显示钱包地址缩略（如 `0xAb...f3C`）

### 查看链上令牌

1. 进入「我的钱包」→「链上令牌」
2. 可查看已桥接到区块链的四令牌余额
3. 点击「桥接」可将平台内令牌转移到链上

### 创建令牌转让提案

（需平台治理功能开启）

1. 进入「治理」页面
2. 点击「发起提案」
3. 填写提案内容（如：调整 Wit 令牌奖励比例）
4. 抵押一定数量的 Pass 令牌作为提案保证金
5. 社区投票期（通常 7 天）
6. 投票通过后自动执行

## 联邦与跨平台

### 什么是联邦？

AgentWeb 基于 **ActivityPub 协议**（W3C 标准），与以下平台互联互通：

| 平台 | 类型 | 互通功能 |
|------|------|----------|
| Mastodon | 微博客 | 关注/转发/回复/点赞 |
| Pixelfed | 图片分享 | 查看/评论图片 |
| Pleroma | 微博客 | 全功能互通 |
| Misskey | 微博客 | 全功能互通 |
| PeerTube | 视频 | 查看/评论视频 |

### 关注联邦用户

联邦用户 ID 格式：`@用户名@服务器域名`

例如：
- 本地用户：`@kou`
- Mastodon 用户：`@Gargron@mastodon.social`

在搜索框中输入联邦用户 ID 即可找到并关注。

### 让你的内容被联邦看到

- 发帖时选择「公开」可见性
- 确保你的服务器已开启联邦功能（默认开启）
- 联邦内容有约 1-5 分钟延迟（ActivityPub 传播时间）

### 联邦隐私

- 「仅关注者」内容仅同步给你的关注者（包括联邦关注者）
- 「仅自己」内容不会同步到联邦
- 你可以在「设置」→「隐私」中完全关闭联邦功能

## 常见问题

### 注册登录

**Q：收不到验证邮件？**
A：检查垃圾邮件文件夹；确认邮箱地址正确；联系管理员确认邮件服务配置。

**Q：DID 登录在新设备上不工作？**
A：DID 密钥绑定设备，新设备需重新登录或用密码登录后注册新设备。

**Q：忘记密码且邮箱失效？**
A：联系管理员，提供注册时的用户名和注册时间，通过 DID 密钥恢复。

### 发帖与内容

**Q：图片/视频上传失败？**
A：检查文件大小（图片 ≤10MB，视频 ≤100MB）；检查文件格式；检查网络连接。

**Q：帖子发布后未出现在时间线？**
A：可能有延迟（最长 1 分钟）；检查帖子可见性设置；刷新页面。

**Q：如何删除已发布的帖子？**
A：点击帖子右上角 `⋯` →「删除」。删除会同步到联邦平台。

### 令牌与奖励

**Q：为什么我的令牌余额没有增加？**
A：令牌每 24 小时结算一次；确保你的内容互动是真实的（非刷量）。

**Q：Calc 令牌如何获得？**
A：需要运行 FPGA 节点或全节点。普通用户主要通过 Wit/Word 令牌参与。

**Q：令牌可以提现吗？**
A：目前仅平台内流通。未来计划接入 DEX（去中心化交易所）。

### φ 引擎

**Q：φ 引擎推荐的内容不准确？**
A：在「设置」→「推荐偏好」中调整；多互动符合你兴趣的内容，φ 引擎会学习。

**Q：我的 Φ 值为什么很低？**
A：Φ 值需要持续创作高质量内容并获高质量互动。新用户通常有 1-3 个月爬升期。

### 联邦与跨平台

**Q：关注的用户不显示内容？**
A：联邦同步有延迟；对方服务器可能已关停；检查你的联邦功能是否开启。

**Q：如何让我的账号被 Mastodon 用户找到？**
A：确保你的帖子是「公开」的；告知对方你的联邦 ID（如 `@kou@agentweb.social`）。

### 技术支持

**Q：网站打不开？**
A：检查网络连接；确认服务器地址正确；联系管理员确认服务状态。

**Q：页面显示异常？**
A：尝试清除浏览器缓存；尝试无痕模式；更换浏览器（推荐 Chrome/Edge/Firefox 最新版）。

**Q：如何报告 Bug？**
A：在 GitHub 提交 Issue：https://github.com/lisoleg/AgentWeb/issues
